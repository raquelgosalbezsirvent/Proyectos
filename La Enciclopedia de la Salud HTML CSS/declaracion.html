Identificación del idioma principal

Esta página web está diseñada para garantizar que todos los elementos están correctamente identificados, incluyendo el idioma principal del documento. El atributo lang="es" está configurado en la etiqueta <html> para que los navegadores, lectores de pantalla y motores de búsqueda puedan interpretar correctamente el contenido en español.

Etiquetado semántico

Se ha hecho uso correcto de las etiquetas HTML5 semánticas como <header>, <main>, <nav>, y <footer>, para proporcionar una estructura clara de la página. Esto facilita la navegación y mejora la accesibilidad para personas que utilizan tecnologías de asistencia, como lectores de pantalla. También ayuda a los motores de búsqueda a entender mejor la estructura del contenido.

Texto alternativo de las imágenes

Todas las imágenes presentes en esta página incluyen un atributo alt que describe de manera concisa el contenido visual. Esto garantiza que los usuarios que no pueden ver las imágenes, ya sea por el uso de un lector de pantalla o por problemas de conexión, puedan comprender el contenido de la página.

Mecanismos de navegación implementados

Se han implementado diversos mecanismos de navegación accesibles:

  • Enlaces con descripciones claras. odos los enlaces están correctamente etiquetados para describir el destino o acción a realizar, sin ambigüedades como "haz clic aquí".
  • Menús de navegación accesibles. Los menús de navegación están diseñados para ser accesibles mediante teclado, permitiendo a los usuarios navegar por los enlaces con la tecla Tab y seleccionarlos con Enter.
  • Saltar al contenido principal. Se ha añadido un enlace para saltar al contenido de la página principal al principio y al final de la página para usuarios de teclado y lectores de pantalla, que les permite saltar directamente sin tener que navegar por el menú de navegación.

Otras características de accesibilidad

Algunas de las características adicionales implementadas para mejorar la accesibilidad incluyen:

  • Tablas accesibles. Las tablas utilizan adecuadamente las etiquetas <th> para los encabezados de fila y columna, y los elementos <caption> para describir el contenido de la tabla, lo que mejora la comprensión del contenido para los usuarios de tecnologías de asistencia.
  • Formularios accesibles. Se han proporcionado etiquetas <label> para cada campo de los formularios de la web, lo cual es esencial para que los usuarios que usan tecnología asistiva comprendan la función de cada campo.
  • Abreviaturas con descripción. Cada abreviatura que aparece en la web posee una etiqueta <title> que permite mostrar una descripción de la abreviación utilizada.
  • Uso de títulos estructurados. Los títulos (<h1>, <h2>, etc.) se han utilizado de manera jerárquica y lógica para organizar la información y hacerla más accesible para usuarios de lectores de pantalla.
